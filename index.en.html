TradeSizer — Crypto Position & Stop-Loss Calculator

An application for calculating position size and stop-loss based on ATR. Includes a volatility analysis tool using NATR.

  • Manual input of ATR, asset price, risk, and multiplier
  • Automatic calculation of position size and stop-loss
  • NATR-based volatility indicator with adjustable thresholds
